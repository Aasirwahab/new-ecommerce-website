# Kimi Agent Premium Website

Next.js 16 App Router project using React 19, TypeScript, Tailwind CSS, and shadcn-style UI components.

## Requirements

- Node.js 20.19+
- pnpm 10.23+

## Development

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000.

## Scripts

```bash
pnpm dev
pnpm build
pnpm start
pnpm lint
```

## Project Structure

- `src/app/layout.tsx` - App Router root layout and metadata.
- `src/app/page.tsx` - Main route with app providers.
- `src/screens/Home.tsx` - Home page composition.
- `src/sections` - Page sections.
- `src/components/ui` - Reusable UI components.
- `public` - Images and video assets.
